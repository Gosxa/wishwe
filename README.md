# WishWe

**WishWe** is a social event platform for planning things together with friends. Users create **wishes** — open-ended ideas like "let's go hiking sometime" — or **plans** — scheduled events with a date, time, and participants. Friends see each other's events in a feed, join in, and get notified when something is happening.

## Features

- 🎯 **Wishes & Plans** — post an open idea or schedule a concrete event with participation tracking and automatic expiration
- 👥 **Friends** — friend requests, invite links, and friend search
- 🔒 **Visibility control** — events can be visible to friends only or to friends of friends
- 🔔 **Notifications** — in-app notifications and scheduled reminders
- 🔗 **Invite & deep links** — share an event link
- 🔑 **Auth** — email sign-up with verification and Google OAuth
- 🖼️ **Profiles** — avatars with in-browser cropping
